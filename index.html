The Reading List

TRL is a curated, monthly list of books to read that help you improve yourself—from marketing, to business, to non-fluffy self-help.

These are the books that I’m reading that you might want to read as well. You’re busy, but love to read, I want to help make that easier by suggesting some choice titles.

This tiny project was created by Paul Jarvis, an author and entrepreneur, but more importantly, an avid reader.
